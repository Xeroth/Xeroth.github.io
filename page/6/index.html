2016-01-15

Docker 常用命令

查看容器ip

docker inspect 6a |grep IPAddress | cut -d '"' -f 4

运行一个容器, 并进入容器的 shell 中做交互

docker run -it ubuntu /bin/bash

删除所有镜像(正在运行的删除不了)

docker rm `docker ps -a -q`

进入一个正在运行的容器

docker exec -ti <container_id> bash

删除一个容器, 同时移除挂载在上面的 volume

docker rm -v <container_id>

如果容器有挂载的, 删除容器时候没移除 volume, 会在本地留下很多挂载目录…强迫症会很难受

查看镜像历史

docker history <container_id>

查看容器目录的在本地的挂载点
