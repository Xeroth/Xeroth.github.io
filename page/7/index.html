解决一些电脑问题

windows7 内部版本7601,此windows副本不是正版

下载小马win7激活工具, 解压, 运行

ubuntu 终端中文乱码

ubuntu iterm 连接之后, 终端无法输入中文 , vim 里面的中文都是乱码, ubuntu tty 显示中文为乱码

解决方法:

添加中文字符编码：
$sudo vim /var/lib/locales/supported.d/local
#添加下面的中文字符集, 一开始是只有en_US.UTF-8 UTF-8这一行的
zh_CN.GBK GBK
zh_CN.GB2312 GB2312
zh_CN.GB18030 GB18030
使其生效：
$sudo dpkg-reconfigure locales
然后重启, 或者重新登录终端就 ok 了

2016-02-18

Django第二天

创建数据库

相关文件是mysite/settings.py , 默认用的是 SQLite . 我用的 mysql(之前在本地安装好的)

先在 mysql 中创建库,用户

mysql> CREATE DATABASE polls DEFAULT CHARSET=utf8;
Query OK, 1 row affected (0.00 sec)

mysql> GRANT SELECT, INSERT, UPDATE, DELETE, CREATE, DROP, INDEX, ALTER, CREATE TEMPORARY TABLES, LOCK TABLES ON polls.* TO 'poller'@'127.0.0.1' IDENTIFIED BY 'poller';
Query OK, 0 rows affected (0.00 sec)

修改mysite/settings.py中的数据库设置

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'NAME': 'polls',
        'USER': 'poller',
        'PASSWORD': 'poller',
        'HOST':'127.0.0.1',
        'PORT':'3306',
    }
}

面试技巧

文章目录

面试可以问哪些问题？

  • 岗位具体的工作内容是什么？
  • 在这家公司工作带来的最大的好处和坏处是什么？
  • 福利待遇升职加薪的机制是什么？

自我介绍要提前想好。这样你才能把面试官引导进入你想表达的内容里，这样才能让面试官说出更多帮助你分析了解团队的事情。

hr 肯定会问你之前跳槽的原因, 提前想好

时刻维护好你自己的简历，每次做完一个项目、拿下一个客户、参加一次培训都把这些经历添加到你的简历上面，不要等到找工作的时候再去添加这些内容
