Ansible synchronize

注意几个参数,[default: no]或者有特殊配置的:

  • copy_links 拷贝软连接的真实文件, 而不是软连接
  • delete 在进行同步后是否删除 src 中不存在的文件, 默认情况下, 是会保留 dest 目标文件夹原先就存在的文件
  • existing_only ?
  • mode [ default: push ]本地或者委托到目标机
