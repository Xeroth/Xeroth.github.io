如何用hexo-github搭建静态博客

一 安装nodes

http://mac-osx-for-newbie-book.kejyun.com/software/softwareWebDeveloperNodeJS.html

二 安装初始化配置 hexo

初始化

生成静态文件

本地运行服务查看,访问 http://localhost:4000

三 配置 github page

注册 github 账号
新建一个仓库, 仓库名: yourusername.github.io (必须是这个格式)

四 部署到github

打开 _config.yml 文件, 修改:

deploy:
       type: git
    repository: git@github.com:yourusername/yourusername.github.io.git
    branch: master

然后部署

如果提示：

$ hexo new “文章标题”

$ hexo g -d
```

参考: https://hexo.io/zh-cn/docs/
